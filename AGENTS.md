# AGENTS.md — Awesome Open Source LLMs

## Purpose

This repository is a curated registry of open-source large language models, training frameworks, and evaluation resources.

## Scope

Cover resources for discovering and working with open-source LLMs:

- Open-source foundation and fine-tuned language models.
- Model hubs, leaderboards, and evaluation benchmarks.
- Training, fine-tuning, and alignment frameworks.
- Licensing and governance resources for open-source models.

## Style

- Follow the same format as the other `awesome-*` repositories in the Enterprise AI Atlas organization.
- Label official/vendor resources as `Official` and third-party resources as `Community`.
- Prefer official documentation and links.

## Maintenance

- Run `./scripts/validate-links.sh` before finishing any edit.
- Place entries under the model family, framework, or benchmark category they primarily target.

## Important reminders

- Do not provide legal advice.
- Verify that links resolve and are current.
